01 Wordmark
ZYLO Wordmark — Usage

A geometric construction — Z and Y as diagonal polygon strokes, L as a right-angle form, O as a perfect circle with inner ring. Always use the SVG source file. Never rasterize below 1× and never set in any font.

White on #060606

Wordmark Rules
✓  Always use the SVG file. Min display size: 120px wide screen / 25mm print.
✓  Clear space: equal to the height of the Y on all four sides.
✗  Never stretch, rotate, outline, shadow, or add effects to the wordmark.
✗  Never set in any system font — the geometry is custom and cannot be reproduced.
✗  Never add "Instruments" beside the wordmark — the full name is the wordmark itself.

Z Mark Rules

Typography Rules
✓  Syne for all headings, labels, buttons, nav — anything uppercase or display.
✓  Inter for all body paragraphs and multi-line descriptive text.
✓  Label tracking: 0.12em minimum. Nav links: 0.06em. Display headlines: -0.02em.
✗  No serif typefaces anywhere in the brand.
✗  Never set body copy in Syne — it is a display face, not readable at paragraph length.
✗  No italic for emphasis — use weight or color instead.

Image Usage Rules
DoAlways match the image to the product's folder. Don't use a ZyloVerb ballet image for a Medrone post.
DoStack two overlays: dark base first (#000 at 45–60%), then product color tint (30–40%). This keeps contrast while letting color read.
DoLet the image dominate. Type should be minimal — product name + one tagline line maximum on atmospheric posts.
DoUse the ZYLO wordmark at low opacity (60–80%) on every post. It's always there, never loud.
Don'tDon't over-tint. If the image is no longer readable through the overlay, the color is too heavy.
Don'tDon't use images from 4_image_ref for product-specific posts — those are general brand atmosphere only.
Don'tDon't add drop shadows, stroke outlines, or borders to images. Images bleed to the edge, always full-bleed.
Don'tDon't mix two products in one image. One post, one product, one color.

15 Design Principles
Core Principles
Recessive

The interface disappears when you work. Dark surfaces, restrained contrast, nothing competing with the audio.

Atmospheric

Evoke sensation over specification. Name the feeling — not the spec sheet.

Precise

2px buttons. 11px labels. One sentence that lands beats a paragraph. Everything as tight as it can be.

Confident

No exclamation marks. No superlatives. State things plainly — the work speaks for itself.

Merch Philosophy

Zylo apparel is minimal, dark, and wearable by anyone who makes music — not a walking ad. Every piece should look like it belongs on a record label, not a software company. Black garments only. White or product-color print only. No gradients, no illustrations, no photography on the garment itself.

Left Chest Icon
Product icon at ~2.5cm. White only. No product color on black garments — the icon reads through tone, not hue.
Center Wordmark
ZYLO wordmark at ~18cm wide. Full chest. Statement piece. The tee-dress treatment — oversized, clean, confident.
Back Text
Optional: product tagline or "zyloinstruments.com" in IBM Plex Mono, 9pt, centered at neck. Low key. No shouting.
Approved Marks for Apparel

Only these marks go on garments. No modified versions, no color fills on dark fabric.

Apparel Rules
DoBlack garments only. Faded black, washed black, charcoal — all fine. The garment is the canvas.
DoWhite print only for standard drops. Single-color screen print or embroidery. No pantone fills.
DoKeep placement architectural: left chest at ~2.5cm, center front at ~18cm wide, back neck text at ~9cm wide.
Don'tNo color fills on the garment — not even product accent colors. That's for digital. Apparel stays tonal.
Don'tNo full-back graphics, no allover prints, no photographic imagery on the garment itself.
